Hallo lezer!

Ik was benieuwd hoe een website in elkaar zat, dit is het resultaat: een hobby "Portfolio". Ookal is het nog niet af.

Al een tijd vind ik het leuk om dingen te ontwerpen, ik teken, ontwerp logo's en bewerk foto's. Hier kan je vinden wat ik zowel heb gemaakt.

Voor mijn komende projecten, zowel voor projecten van school als in mijn vrije tijd, zou ik graag beter willen worden in tekenen op papier en op de computer. HTML en CSS leren stond ook op het progamma, maar zoals je kan zien ben ik daar mee bezig geweest. Ook wil ik een kleine game en animatie maken.
